# Recetas de cocina

Vault de recetas de cocina editado en obsidian.md y exportado a formato web con el plugin [obsidian webpage export](https://github.com/KosmosisDire/obsidian-webpage-export).

[Descarga el repositorio en formato zip](https://github.com/obsidian-pkm-vault/recetas-cocina/archive/refs/heads/main.zip) o clonalo con git `git clone https://github.com/obsidian-pkm-vault/recetas-cocina.git`.

> Para ver la web de recetas [accede aquí](https://obsidian-pkm-vault.github.io/recetas-cocina/recetas/).
